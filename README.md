# TaskFlow — Project Manager

シンプルで直感的なタスク管理アプリ。**バックログ → タスク**の階層構造で整理し、カンバン・ガントチャート・カレンダーの3つのビューでプロジェクトを管理できます。

---

## 🚀 使い方

`index.html` をブラウザで開くだけで動作します。インストールや依存パッケージのセットアップは不要です。

---

## 📦 技術スタック

| 項目 | 内容 |
|------|------|
| UI フレームワーク | React 18.2 (UMD / CDN) |

> ビルドツール不要。単一の `index.html` ファイルで完結しています。

## 🗂️ データ構造

### バックログ
大きなテーマ・機能単位をまとめる親要素です。

```
{
  id:    string   // 一意のID
  name:  string   // バックログ名
  color: string   // 識別カラー (hex)
}
```

### タスク
バックログに紐づく作業単位です。

```
{
  id:     string   // 一意のID
  blId:   string   // 所属バックログのID
  title:  string   // タスク名
  description: string  // 詳細説明
  status: 'todo' | 'inprogress' | 'done'
  start:  string   // 開始日 (YYYY-MM-DD)
  end:    string   // 終了日 (YYYY-MM-DD)
}
```

## 🖼️ ビュー一覧

### カンバン

| 操作 | 方法 |
|------|------|
| タスク追加 | 各カラム下部の「＋ タスク追加」 |
| ステータス変更 | カードをドラッグして別カラムへドロップ |
| タスク編集・削除 | カードをクリック |

- ステータスは **To Do / In Progress / Done** の3列
- バックログカラーのバッジでタスクの所属を識別

### ガントチャート

| 操作 | 方法 |
|------|------|
| タスク追加 | 左パネルの「＋ タスク追加」またはグリッドをドラッグ |
| 日程移動 | バー中央をドラッグ |
| 開始日変更 | バー左端をドラッグ |
| 終了日変更 | バー右端をドラッグ |
| タスク編集 | 左パネルの ✎ ボタン |
| 今日に戻る | ツールバーの「今日」ボタン |
| 期間移動 | ‹ › ボタン（14日単位） |

- バックログ単位でグループ化・折りたたみ可能
- ドラッグ中はツールチップに日付・期間をリアルタイム表示
- 土曜は青・日曜は赤で色分け

### カレンダー

| 操作 | 方法 |
|------|------|
| タスク編集 | ピルをクリック |
| 月移動 | ‹ › ボタン |
| 今月に戻る | 「今月」ボタン |

- タスクは開始日〜終了日の期間中すべての日に表示
- バックログカラーでピルを色分け
- 土曜は青・日曜は赤で色分け

## ⚠️ 期限ステータス

完了済み（Done）以外のタスクに自動適用されます。

| 状態 | 条件 | 表示 |
|------|------|------|
| 期限超過 | 終了日 < 今日 | 赤バッジ・赤ボーダー |
| 今日期限 | 終了日 = 今日 | 黄バッジ・黄ボーダー |
| 期限内 | 終了日 > 今日 | 通常表示 |

```

├── TaskModal        # タスク作成・編集モーダル

├── ConfirmDialog    # 削除確認ダイアログ
└── HelpModal        # 操作説明モーダル
```
